# 钢管切割下料优化：多方法建模与精确线性化

> 针对「钢管切割下料问题」中**非线性前K大加权目标**，设计并对比了 **4 种不同的建模与求解方法**，将非凸目标逐步转化为线性 MILP，实现了精确全局最优求解。

---

## 📦 问题描述

- 原料钢管长 1850 mm，需满足 4 种产品的需求：

| 产品 | 长度   | 需求量 |
|------|--------|--------|
| 1    | 290 mm | 15 根  |
| 2    | 315 mm | 28 根  |
| 3    | 350 mm | 21 根  |
| 4    | 455 mm | 30 根  |

- 切割限制：
  - 每根原料最多切成 5 根产品，余料 ≤ 100 mm
  - 使用的切割模式种类 ≤ 4 种
- 成本规则：
  - 使用次数最多的模式成本 ×1.1，第二多 ×1.2，第三多 ×1.3，第四多 ×1.4
  - **目标**：最小化总加权成本（以“根”为单位）

---

## 🧠 建模方法概览

| 文件 | 方法 | 工具 | 算法类型 | 核心技术 |
|------|------|------|----------|----------|
| `nonlinear.m` | **原始非线性模型 + 遗传算法** | MATLAB `ga` | 启发式 | 直接计算排序后的前四大加权和作为适应度 |
| `convex.m` | **大M排除法 MILP** | MATLAB `intlinprog` | 精确 | 引入 0‑1 指示变量 `z` 和大 M 约束，将前四大提取转化为线性不等式 |
| `QCP.py` | **排序约束 + 二次约束模型** | Python + Gurobi | 精确（非凸） | 直接令 x₁≥x₂≥x₃≥x₄，但含有 xᵢ·aᵢ 二次项 |
| `MILP.py` | **二进制展开完全线性化 MILP** | Python + Gurobi | 精确 | 将整数变量二进制展开，线性化所有二次项，转化为纯 0‑1 线性规划 |

---

## 📐 数学模型

### 符号定义

- 模式索引 i = 1,…,4 (或 1,…,11 在枚举版本中)
- Nᵢ ：第 i 种模式的使用次数（0 ≤ Nᵢ ≤ 30）
- Iᵢ ：是否选用第 i 种模式（0/1）
- A_{ip} ：模式 i 生产产品 p 的数量（p=1,2,3,4，长度依次 290,315,350,455）
- d_p ：产品 p 的需求量 [15,28,21,30]

---

### 1. nonlinear.m – 原始非线性模型（GA 求解）

**目标**（在适应度函数中计算）：
min 1.1·N₍₁₎ + 1.2·N₍₂₎ + 1.3·N₍₃₎ + 1.4·N₍₄₎
其中 N₍₁₎ ≥ N₍₂₎ ≥ N₍₃₎ ≥ N₍₄₎ 是 N 的降序排列（前四大）。

**约束**（线性部分）：
- ∑ Iᵢ ≤ 4
- Nᵢ ≤ 30 Iᵢ ， Nᵢ ≥ Iᵢ
- ∑ᵢ Nᵢ A_{ip} ≥ d_p  （p=1..4）

目标函数包含排序，无法写成线性表达式，因此采用遗传算法直接计算适应度。

---

### 2. convex.m – 大M排除法 MILP

**新增变量**：
- z₁ᵢ, z₂ᵢ, z₃ᵢ ∈ {0,1} ：分别标记“是最大值”、“属于前两大”、“属于前三大”
- t₁, t₂, t₃, t₄ ≥ 0 ：第1~第4大的使用次数

**目标**：
min 1.1 t₁ + 1.2 t₂ + 1.3 t₃ + 1.4 t₄

**约束**（除原始约束外）：
- ∑ z₁ᵢ = 1 ， ∑ z₂ᵢ = 2 ， ∑ z₃ᵢ = 3
- z₁ᵢ ≤ z₂ᵢ ≤ z₃ᵢ
- 若 z₁ᵢ = 1 则 Nᵢ ≥ Nⱼ ∀j  （大M实现）
- 前四大提取（线性化 max）：
t₁ ≥ Nᵢ ∀i
t₂ ≥ Nᵢ - M z₁ᵢ ∀i
t₃ ≥ Nᵢ - M z₂ᵢ ∀i
t₄ ≥ Nᵢ - M z₃ᵢ ∀i
其中 M = 100（远大于30）

由于目标极小化，t 会恰好取到对应的 max 值。

---

### 3. QCP.py – 二次约束模型

**变量**：
- xᵢ ∈ [0,30] 整数：第 i 种模式的使用次数（i=1..4，强制排序 x₁≥x₂≥x₃≥x₄）
- aᵢ, bᵢ, cᵢ, dᵢ ∈ [0,5] 整数：模式 i 生产的四种产品数量

**目标**：
min 1.1 x₁ + 1.2 x₂ + 1.3 x₃ + 1.4 x₄

**约束**：
- 需求满足（含二次项）：
∑ᵢ xᵢ aᵢ ≥ 15
∑ᵢ xᵢ bᵢ ≥ 28
∑ᵢ xᵢ cᵢ ≥ 21
∑ᵢ xᵢ dᵢ ≥ 30
- 模式可行性：
aᵢ + bᵢ + cᵢ + dᵢ ≤ 5
1750 ≤ 290 aᵢ + 315 bᵢ + 350 cᵢ + 455 dᵢ ≤ 1850
- 排序约束：
x₁ ≥ x₂ ≥ x₃ ≥ x₄

该模型为 **MIQCP**（混合整数二次约束规划），可直接用 Gurobi 求解（需许可证）。

---

### 4. MILP.py – 二进制展开线性化

将 `QCP.py` 中的二次项 **完全线性化**。

**核心技术**：
- 将整数变量 xᵢ 用 5 位二进制表示：xᵢ = ∑_{k=0}⁴ 2^k X_{ik}，X_{ik}∈{0,1}
- 将 aᵢ,bᵢ,cᵢ,dᵢ 用 3 位二进制表示：aᵢ = ∑_{l=0}² 2^l A_{il}，A_{il}∈{0,1}
- 乘积 X_{ik}·A_{il} 用辅助 0‑1 变量 P 线性化：
P ≤ X_{ik}
P ≤ A_{il}
P ≥ X_{ik} + A_{il} - 1
- 最终所有约束（需求、模式可行性）均化为线性表达式。

**结果**：得到纯 **0‑1 线性规划**，可用任何 MILP 求解器求解，无需处理二次项。

---

## 📊 结果对比

所有精确方法均稳定收敛到同一全局最优目标值 **21.5**。

- **GA 启发式**：在多数运行中能找到 21.5，但偶尔会偏离，且每次运行结果可能不同。
- **大M MILP**：几分钟内精确求解，结果完全可复现。
- **二进制展开 MILP**：变量数较多，但利用 Gurobi 可在数秒内精确求解。

## 🚀 如何运行

### MATLAB 环境
1. 安装 **MATLAB R2020a+** 及 **Optimization Toolbox**
2. 运行 `nonlinear.m` 或 `convex.m`

### Python 环境
```bash
pip install gurobipy
python QCP.py      # Gurobi 可求解二次约束（需完整许可证）
python MILP.py     # 纯 MILP，学术许可即可

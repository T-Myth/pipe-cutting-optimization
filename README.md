# 钢管切割下料优化：多方法建模与精确线性化

> 针对「钢管切割下料问题」中**非线性前K大加权目标**，设计并对比了 **4 种不同的建模与求解方法**，将非凸目标逐步转化为线性 MILP，实现了精确全局最优求解。

---

## 问题描述

- 原料钢管长 1850 mm，需满足 4 种产品的需求：

| 产品 | 长度 (mm) | 需求量 (根) |
|:----:|:---------:|:-----------:|
|  1   |    290    |     15      |
|  2   |    315    |     28      |
|  3   |    350    |     21      |
|  4   |    455    |     30      |

- **切割限制**：
  - 每根原料最多切成 5 根产品，余料 ≤ 100 mm
  - 使用的切割模式种类 ≤ 4 种

- **成本规则**：
  - 使用次数最多的模式成本为原成本的 **1.1 倍**，第二为 **1.2 倍**，第三为 **1.3 倍**，第四为 **1.4 倍**
  - 目标：最小化总加权成本（以“1”为单位成本）

---

## 建模方法概览

| 文件 | 方法 | 工具 | 算法类型 | 核心技术 |
|------|------|------|----------|----------|
| `nonlinear.m` | **原始非线性排序模型 + 遗传算法** | MATLAB `ga` | 启发式 | 直接计算排序后的前四大加权和作为适应度 |
| `MILP_bigM.m` | **大M排除法 MILP** | MATLAB `intlinprog` | 精确 | 引入 0‑1 指示变量 `z` 和大 M 约束，将前四大提取转化为线性不等式 |
| `MIQCP.py` | **排序约束 + 二次约束模型** | Python + Gurobi | 精确（非凸） | 直接令 $x_1\ge x_2\ge x_3\ge x_4$，但含有 $x_i\cdot a_{ip}$ 二次项 |
| `MILP_binary.py` | **二进制展开完全线性化 MILP** | Python + Gurobi | 精确 | 将整数变量二进制展开，线性化所有二次项，转化为纯 0‑1 线性规划 |

---

### 1. nonlinear.m – 原始非线性排序模型
在建立具体的数学模型之前，首先需要找出所有满足生产限制的切割模式。  
根据“每根钢管最多切成 5 根产品、余料 ≤100 mm”的规则，通过穷举每种产品可能的数量组合，筛选出全部可行模式。  

```matlab
% 先找到所有可能的模式，遍历搜索
% 定义约束条件
A = [1, 1, 1, 1; 
    290, 315, 350, 455;
    -290,-315,-350,-455];
b = [5; 1850;-1750];
% 初始化切割模式矩阵
results_matrix = [];
% 循环遍历所有结果
for x1 = 0:5,for x2 = 0:5,for x3 = 0:5,for x4 = 0:5
    x = [x1 x2 x3 x4]'; % 组合向量
    % 检查是否满足所有约束条件
    if all(A*x <= b)
        results_matrix = [results_matrix;x'];
    end
end; end; end; end
```
最终得到 **11 种切割模式**，作为后续建模的基础。
**完整优化模型**：
- **变量**：
  - $N_i$：第 $i$ 种模式的使用次数（ $0 \le N_i \le 30$ ）
  - $I_i$：是否选用第 $i$ 种模式（ $I_i \in $\{0,1\} ）
  - $a_{ip}$：模式 $i$ 生产产品 $p$ 的数量（ $p=1,2,3,4$，长度依次为 290、315、350、455 mm）
  - $d_p$：产品 $p$ 的需求量 $[15, 28, 21, 30]$

- **目标函数**：

 $$\min \quad 1.1\, N_{(1)} + 1.2\, N_{(2)} + 1.3\, N_{(3)} + 1.4\, N_{(4)}$$

其中 $N_{(1)} \ge N_{(2)} \ge N_{(3)} \ge N_{(4)}$ 是 $N$ 的降序排列（前四大）。

- **约束条件**：

$$\sum_{i=1}^{11} I_i \le 4$$

$$\sum_{i=1}^{11} N_i \cdot a_{ip} \ge d_p \quad p=1,2,3,4$$

$$N_i \le 30 I_i \quad \forall i$$

$$I_i \le N_i \quad \forall i$$

> **建模要点**：通过 $M=30$ 的大 M 技巧，将逻辑条件 $I_i=1 \Leftrightarrow N_i\ge 1$， $I_i=0 \Leftrightarrow N_i=0$ 等价转化为两条线性不等式，使得整个约束集保持线性。但因目标函数包含排序操作，无法线性表达，故使用遗传算法（GA）求解。

### 2. MILP_bigM.m – 大M排除法 MILP

**完整优化模型**：

- **新增变量**：
  - $z_{1i}, z_{2i}, z_{3i} \in$ \{0,1\}：分别标记“是最大值”、“属于前两大”、“属于前三大”，是则为1，否则为0
  - $t_1, t_2, t_3, t_4 \ge 0$：第1~第4大的使用次数

- **目标函数**：

$$\min \quad 1.1 t_1 + 1.2 t_2 + 1.3 t_3 + 1.4 t_4$$

- **约束条件**：

**原始约束**

$$\sum_{i=1}^{11} I_i \le 4$$

$$\sum_{i=1}^{11} N_i a_{ip} \ge d_p \quad p=1,2,3,4$$

$$N_i \ge I_i,\; N_i \le 30 I_i \quad \forall i$$

**排序指示变量定义**

$$\sum_i z_{1i} = 1,\ \sum_i z_{2i} = 2,\ \sum_i z_{3i} = 3$$

$$z_{1i} \le z_{2i} \le z_{3i} \quad \forall i$$

**最大值标记（大M）**

$$N_i \ge N_j - M(1 - z_{1i}) \quad \forall i,j \qquad (M=100)$$

**前四大线性提取（排除法）**

$$t_1 \ge N_i \quad \forall i$$

$$t_2 \ge N_i - M z_{1i} \quad \forall i$$

$$t_3 \ge N_i - M z_{2i} \quad \forall i$$

$$t_4 \ge N_i - M z_{3i} \quad \forall i$$

> **建模要点**：引入 0‑1 指示变量，通过大 M 约束给出正确排序，再利用极小化目标使得辅助变量 $t_k$ 自动等于排除后的最大值，从而实现前四大加权和的精确线性化。模型为 MILP，可用 `intlinprog` 求解。

---

### 3. MIQCP.py – 二次约束模型

**完整优化模型**：

- **变量**：
  - $x_i \in [0,30]$ 整数：第 $i$ 种模式的使用次数（$i=1..4$，强制降序）
  - $a_i, b_i, c_i, d_i \in [0,5]$ 整数：模式 $i$ 生产的四种产品数量

- **目标函数**：

$$
\min \quad 1.1 x_1 + 1.2 x_2 + 1.3 x_3 + 1.4 x_4
$$

- **约束条件**：
  
$$\sum_{i=1}^{4} x_i a_i \ge 15$$

$$\sum_{i=1}^{4} x_i b_i \ge 28$$

$$\sum_{i=1}^{4} x_i c_i \ge 21$$

$$\sum_{i=1}^{4} x_i d_i \ge 30$$

$$a_i + b_i + c_i + d_i \le 5 \quad \forall i$$

$$1750 \le 290a_i + 315b_i + 350c_i + 455d_i \le 1850 \quad \forall i$$

$$x_1 \ge x_2 \ge x_3 \ge x_4$$
  

> **建模要点**：将模式设计与次数选择同时优化，利用预设的排序约束 $x_1\ge\cdots\ge x_4$ 避免后续排序。但需求约束中包含 $x_i a_i$ 等二次项，因此模型为混合整数二次约束规划（MIQCP），可直接用 Gurobi 求解。

---

### 4. MILP_binary.py – 二进制展开线性化 MILP

将 `MIQCP.py` 中的二次项 **完全线性化**，得到纯 0‑1 线性规划。

**线性化方法**：

- 对整数变量 $x_i$（0~30）用 5 位二进制表示：

$$x_i = \sum_{k=0}^{4} 2^k X_{ik},\quad X_{ik}\in\{0,1\}$$

- 对零件数量 $a_i,b_i,c_i,d_i$（0~5）用 3 位二进制表示：
  
$$a_i = \sum_{l=0}^{2} 2^l A_{il},\quad A_{il}\in\{0,1\}$$

- 乘积 $X_{ik} \cdot A_{il}$ 用辅助 0‑1 变量 $P=X_{ik} \cdot A_{il}$ 线性化：
  
$$P \le X_{ik}$$
  
$$\quad P \le A_{il}$$
  
$$\quad P \ge X_{ik} + A_{il} - 1$$
  
- 所有原二次约束（需求、长度）均转化为线性表达式。

**最终模型**：纯 0‑1 线性规划，可用任何 MILP 求解器求解，无需处理二次项。

> **建模要点**：通过二进制分解将整数变量的乘积转化为多个二进制变量乘积，再利用经典的线性化技巧消除所有非线性，得到一个纯粹的整数线性规划。变量数增加，但模型适应性极强，适用于各类 MILP 求解器。

---

## 结果对比

所有精确方法均稳定收敛到同一全局最优目标值 **21.5**。

- **GA 启发式**：在多数运行中找到 21.5，但偶尔偏离，且运行结果有波动。
- **大M MILP**：精确求解，结果完全可复现。
- **二进制展开 MILP**：变量数多但利用 Gurobi 可在数秒内求解。

## 如何运行

### MATLAB 环境
1. 安装 **MATLAB R2020a+** 及 **Optimization Toolbox**
2. 运行 `nonlinear.m` 或 `MILP_bigM.m`

### Python 环境
```bash
pip install gurobipy
python MIQCP.py      # Gurobi 可求解二次约束（需完整许可证）
python MILP_binary.py     # 纯 MILP，学术许可即可
